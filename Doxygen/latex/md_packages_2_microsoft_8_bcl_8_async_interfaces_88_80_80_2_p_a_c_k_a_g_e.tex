As of C\# 8, the C\# language has support for producing and consuming asynchronous iterators. The library types in support of those features are available in .NET Core 3.\+0 and newer as well as in .NET Standard 2.\+1. This library provides the necessary definitions of those types to support these language features on .NET Framework and on .NET Standard 2.\+0. This library is not necessary nor recommended when targeting versions of .NET that include the relevant support.\doxysection{Key Features}\label{md_packages_2_microsoft_8_bcl_8_async_interfaces_88_80_80_2_p_a_c_k_a_g_e_autotoc_md37}
