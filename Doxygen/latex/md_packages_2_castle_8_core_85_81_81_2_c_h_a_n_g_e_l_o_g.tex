\item fix -\/ whenever custom IProxy\+Generation\+Hook is used, user should account for System.\+Object\textquotesingle{}s members being now passed to Should\+Intercept\+Method and Non\+Virtual\+Member\+Notification methods and if necessary update the code to handle them appropriately.
